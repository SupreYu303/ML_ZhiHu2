# CBR-RBR 混合推理工程校核报告

## 1. CBR-RBR 模块目的

CBR-RBR 混合推理机制将历史相似案例检索（Case-Based Reasoning）与显式工程规则（Rule-Based Reasoning）相结合，一方面利用 CBR 为预测结果提供相似工程案例依据，另一方面利用 RBR 对预测位移、埋深、巷道宽度及支护参数进行规则化校核。该机制不替代预测模型，而是对预测结果进行工程合理性复核。

## 2. CBR 案例推理方法

- 案例库: 全部 176 条历史样本
- 特征空间: 8 个输入特征经 StandardScaler 标准化
- 距离度量: 欧氏距离
- 相似度: similarity = 1 / (1 + distance)
- 检索: 对每个测试样本检索 Top-5 相似案例

## 3. RBR 规则推理方法

### 3.1 位移风险等级

| 预测位移 (mm) | 风险等级 |
|:-----------:|:------:|
| < 50 | 低风险 |
| 50-100 | 中风险 |
| 100-150 | 较高风险 |
| >= 150 | 高风险 |

### 3.2 支护参数校核规则

- 锚杆支护面积 <= 0.55: 建议复核锚杆支护强度
- 锚索密度 <= 0.8: 建议复核锚索补强措施
- 预测位移 >= 100 且锚索密度 <= 1.0: 建议加强锚索支护
- 预测位移 >= 100 且锚杆支护面积 <= 0.64: 建议提高锚杆支护参数

### 3.3 工程地质条件校核

- 埋深 >= 700: 关注深部高地应力影响
- 巷道宽度 >= 6.5: 复核顶板控制效果
- 顶板强度 <= 30: 加强顶板稳定性复核
- 煤层强度 <= 5: 关注围岩变形风险

## 7. 结论

CBR-RBR 混合推理机制将历史相似案例检索与显式工程规则相结合，一方面利用 CBR 为预测结果提供相似工程案例依据，另一方面利用 RBR 对预测位移、埋深、巷道宽度及支护参数进行规则化校核。该机制能够提高机器学习预测结果的工程可解释性，并为支护参数调整和现场风险复核提供辅助依据。
